OPERADORA DE TV SATELITAL

Como parte del equipo BAU, desempeñé un rol clave en la actualización de diversos servicios, optimizando los tiempos de respuesta de procesos críticos. Además, participé en el desarrollo de un nuevo módulo de cobros desde cero, mejorando la eficiencia y reduciendo dependencias externas. El proyecto se basó en una arquitectura SOA, utilizando la suite de Oracle y JDeveloper como herramientas principales.

Mi rol consistía en el desarrollo de nuevas funcionalidades, resolución de incidencias, optimización de procesos y actualización de servicios existentes. Utilicé tecnologías como Java, Oracle y JDeveloper para garantizar la calidad y el rendimiento de las aplicaciones.

PROFESIONAL

BILLETERA VIRTUAL

Actualmente, colaboro en el desarrollo de una billetera virtual basada en una arquitectura de microservicios. Utilizo Java 21 y Spring Boot para implementar nuevas funcionalidades relacionadas con pagos y transferencias, garantizando una experiencia de usuario fluida y segura.

Como desarrollador backend, soy responsable de implementar las funcionalidades asignadas, realizar pruebas unitarias y asegurar la integración del código con el resto de los microservicios para construir soluciones escalables y robustas.

PROFESIONAL

LOG IN

En este proyecto, formé parte del equipo de desarrollo, colaborando en la creación de un sistema de autenticación de usuarios. Utilizamos un conjunto de tecnologías modernas, incluyendo HTML, CSS, React, Node.js y MongoDB, para construir una solución robusta y escalable.

Me enfoqué en el desarrollo de la base de datos MongoDB y la implementación de la lógica de autenticación utilizando tecnologías front-end como HTML, CSS y React. Además, colaboré en la configuración del servidor backend con Node.js.
